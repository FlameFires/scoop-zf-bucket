
# scoop-zf-bucket

我的个人 Scoop bucket，由 zerofire（也称 zf）维护。

## 使用方法

```powershell
scoop bucket add zf https://github.com/FlameFires/scoop-zf-bucket.git
scoop install zf/centbrowser
```

## 可用软件

| 名称 | 版本 | 描述 |
|------|------|------|
| centbrowser | 5.2.1168.83 | Cent Browser - 基于 Chromium 的功能丰富的浏览器 |
| catsxpbrowser | 148.6.5.5.7 | Catsxp Browser - 基于 Chromium 的隐私浏览器（便携版） |
